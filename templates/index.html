Website URL eingeben

Geben Sie die URL der WordPress-Website ein, die Sie als statische Kopie herunterladen möchten.

Funktionen:

  • ✓ Vollständige Website-Erfassung
  • ✓ Automatischer Asset-Download (Bilder, CSS, JS)
  • ✓ Link-Konvertierung für lokale Navigation
  • ✓ ZIP-Export der kompletten Site
  • ✓ Live-Fortschrittsanzeige

Hinweise:

  • • Der Scraping-Prozess kann je nach Website-Größe mehrere Minuten dauern
  • • Nur öffentlich zugängliche Inhalte werden erfasst
  • • Kontaktformulare werden deaktiviert
